# FilterGS: Traversal-Free Parallel Filtering and Adaptive Shrinkage for Large-scale LoD 3D Gaussian Splatting

[![Website](https://img.shields.io/badge/Website-Live-brightgreen)](https://xenon-w.github.io/FilterGS.github.io/)
[![GitHub](https://img.shields.io/badge/GitHub-Repository-blue)](https://github.com/xenon-w/FilterGS.github.io)

## Overview

FilterGS is a novel approach for large-scale Level-of-Detail (LoD) 3D Gaussian Splatting that addresses critical performance bottlenecks through:

- **Traversal-Free Parallel Filtering**: Efficient selection mechanism without tree traversal
- **Adaptive Shrinkage Strategy**: Scene-adaptive Gaussian shrinkage to minimize redundancy
- **State-of-the-Art Performance**: Achieves superior rendering speeds while maintaining competitive visual quality

## Key Features

- Parallel filtering mechanism with two complementary filters
- Opacity-based scaling for Gaussian-tile pair optimization
- Efficient processing of large-scale 3D scenes
- Real-time rendering performance
